# Hangman

A retro, NES-styled take on classic Hangman built with vanilla JavaScript, SCSS, and NES.css. Guess the hidden word one letter at a time — using the on-screen keyboard or your own — before you run out of guesses or the 60-second timer hits zero. Correct and wrong letters are colour-coded and lock as you go, game logic is fully unit-tested with Jest, and each round pulls a random word from a bundled word list.
